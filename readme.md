# Piscine C++ 

Ce dépôt regroupe les projets réalisés durant la **Piscine C++** à 42.  
La piscine a pour objectif d’introduire progressivement le langage **C++** et les principes de la **programmation orientée objet**, en s’appuyant sur une série de modules.

Chaque module se concentre sur des notions spécifiques du langage.

---

Chaque module contient plusieurs exercices indépendants, chacun visant un aspect précis du C++.

---

## Compilation et exécution

Chaque exercice peut être compilé simplement via :

```bash
make
```


Au cours de la piscine, les notions suivantes ont été travaillées :

 - Syntaxe et bases du C++

 - Classes, objets et encapsulation

 - Constructeurs, destructeurs

 - Héritage et polymorphisme

 - Classes abstraites et interfaces

 - Gestion de la mémoire

 - Exceptions

 - Templates

 - Containers et algorithmes de la STL
